thirdparty.xml

< Back

This file contains the locations of third party software which has been added into the framework, and allows them to be used with the autoloader. The elements (under the root element) are the lowercase versions of class names that can be instantiated, and the content of each element is the filename of the PHP file where this class can be found.

To override the settings, first copy the file from sys/config/ to app/config/, and modify the copy.

Default File Contents

<?xml version="1.0" encoding="utf-8"?>

<!-- Each child of the thirdparty root node should be the name of the class,
and the value the file to be included, relative to the thirdparty folder -->

<thirdparty>
  <phpmailer>phpmailer/class.phpmailer.php</phpmailer>
  <fpdf>fpdf/fpdf.php</fpdf>
</thirdparty>
