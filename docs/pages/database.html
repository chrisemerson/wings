Database Class

< Back

This class handles database connections, including reading the config, switching between different database drivers as necessary, and providing functions to deal with transactions if they are required.

As an abstract class, it cannot be instantiated directly, but both the Schema class and the DirectQuery library class extend this class in order to offer their functionality.

Basic Structure

class Dispatcher {
  protected $dbConn;
  protected $strTablePrefix;

  protected function __construct ()
  private function openDBConn ()
  public function startTransaction ()
  public function commit ()
  public function rollback ()
}//class
