This file is used by the Convert class as the source of all the conversion rates used. The file consists of several groups, which represent conversion types (such as 'Distance' or 'Speed'), each of which contains many units that can be converted between within that group.

Each group is constructed so that each value within it is equal (for example, 1 Nautical Mile = 1852 Metres = 6076.11549 Feet). The conversion class handles all converting within the groups beyond that. The largest unit in each case has the value of 1, so that all the other units in the group have a value of more than 1.

Units can share codes with units in other groups, as long as no more than 1 unit code is shared between 2 groups - otherwise the Convert class will not know which conversion rate to use. For example, both Distance and Volume groups contain a 'CM' unit (Centimetres vs Cubic Metres), which is fine because those 2 groups don't share any other unit codes.

The Temperature conversions are hard coded into the Convert class, because all the units involved have different zero points, so cannot be converted in a standard way.

To add more groups or units, first copy this config file into the app/config/ folder, and edit the copied version.
