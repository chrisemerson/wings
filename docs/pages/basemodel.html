BaseModel Class

< Back

This class serves as a base for all the models in the system, and provides them with their ORM functionality. It is abstract and cannot be instantiated on its own. It extends the Schema class, which is reponsible for providing this class with information about tables and columns from the Database.

Along with the Collection class, this class constitutes the ORM functionality for the framework.

The BaseModel class also defines 4 exceptions.

Basic Structure

abstract class BaseModel extends Schema {
  private $arrCurrentData = array();
  private $arrNewData = array();
  private $blnSaved = false;

  public function __construct ($mixPK = null)
  public function __get ($strFieldName)
  public function __set ($strFieldName, $strValue)
  public function __call ($strMethodName, $arrArguments)
  public function save ()
  public function loadFromArray ($arrData)
  public function loadFromDBArray ($arrData)
  private function loadFromPK ($mixPK)
  private function insertIntoDB ()
  private function updateDB ()
  public function delete ()
  protected function isSavedData ()
}//class

class DataMissingException extends Exception {}
class FieldNotFoundException extends Exception {}
class InvalidDataException extends Exception {}
