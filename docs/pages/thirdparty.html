Third Party Classes

< Back

As well as the built in and custom library files, Wings also allows for third party library files to be included and to make use of the framework's autoloading functionality.

Wings includes PHPMailer and FPDF classes within sys/lib/thirdparty/, and additional classes can be added into app/lib/thirdparty. There is also a Config file to edit (thirdparty.xml) to enable the Loader class to determine where the class lives.

If you wish to upgrade or alter any of the included third party classes, you should first copy them from sys/lib/thirdparty/ to app/lib/thirdparty/, and modify the copied version.
