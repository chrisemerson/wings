< Back

This file defines the settings for the database sessions module. Database sessions can be enabled or disabled here, and the rest of the file defines a model to use for the session data, as well as the field names.

To override the settings, first copy the file from sys/config/ to app/config/, and modify the copy.

Default File Contents

<?xml version="1.0" encoding="utf-8"?>

<dbsessions>
  <enabled>0</enabled>

  <model>DBSession</model>

  <fields>
    <id>db_session_id</id>                      <!-- VARCHAR (32) -->
    <accesstime>db_session_access</accesstime>  <!-- DATETIME -->
    <data>db_session_data</data>                <!-- TEXT -->
  </fields>
</dbsessions>
