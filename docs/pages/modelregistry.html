ModelRegistry Class

< Back

This class is responsible for reading the models.xml config file, and converting medel names to pluralised versions, or deriving table names from the model names.

It is used by the Schema class (and in turn the Collection and BaseModel classes) in order to get information on whether something is a model, and which database tables to look in for data.

Basic Structure

class ModelRegistry {
  public static $arrModelInfo;

  public function __construct ()
  private function loadModelInfo ()
  public function isModel ($strModelName)
  public function getPluralisedName ($strModelName)
  public function getTableName ($strModelName)
  public function getModelNameFromPlural ($strPlural)
  private function pluraliseModelName ($strModelName)
  private function convertPluralNameToTableName ($strPluralName)
}//class
