Debug Class

< Back

This class provides several static methods for outputting debug information to the screen, with the ability to turn on or off all output with the ->errors->debugmode setting in app.xml.

Methods & Properties

Debug::e ($mixInput)

This is an alias of the echo statement, but output is only sent if the debugmode setting is on in app.xml.

Debug::vd ($mixInput)

This is an alias of the var_dump() function, but output is only sent if the debugmode setting is on in app.xml.

Debug::pr ($mixInput, $blnReturn = false)

This is an alias of the print_r() function, but output is only sent if the debugmode setting is on in app.xml.
