Library Files

< Back

Library files in wings are additional 'helper' classes that are not part of the main framework, but can be used independently. They are all contained within the lib/ folder, and are autoloaded when called, as long as they are not named the same thing as any of the other components in the framework.

Included files in the framework currently consist of:

  • Authentication
  • Cache (not yet implemented)
  • Captcha
  • Convert
  • Cookie (not yet implemented)
  • CURL
  • DatabaseSession
  • Debug
  • DirectQuery
  • ErrorRegistry
  • FileUpload (not yet implemented)
  • Image (not yet implemented)
  • InputFilter
  • LatLongHelper
  • LogFile
  • PageManager (not yet implemented)
  • Session

Information for the use of these files can be found on the relevant page for each one, and new library files can be added to the app/lib/ folder if required. Library files can be overridden by making a copy within app/lib/ and modifying the copy within that folder.
