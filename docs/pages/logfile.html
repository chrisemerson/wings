LogFile Class

< Back

The LogFile class allows easy logging of information to .log files within the app/logs/ folder.

Log messages are always stamped with the time, date and IP address of the client who generated it.

Methods & Properties

LogFile->__construct ($strLogName) (Constructor)

This opens the logfile with the specified name. If it doesn't exist, it is created and a message at the top of the file is added with the date and time.

LogFile->__destruct () (Destructor)

When the object is destroyed, the log file is closed.

LogFile->logMessage ($strMessage)

Appends the message to the open log file with a timestamp and IP address.
