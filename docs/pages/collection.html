This class serves as the base for all collections in the system, but should not be used directly. Classes that use this base class are created automatically, on the fly. It should never be extended by the user, but left to the framework. Because it is an abstract class, it cannot be instantiated directly. It provides collections with their functionality, including iteration, array access and making it countable.

The Collection class extends Schema and implements the Iterator, Countable and ArrayAccess interfaces.

abstract class Collection extends Schema implements Iterator, Countable, ArrayAccess {
  private $arrMembers = array();
  private $intPosition = 0;
  private $blnJustUnsetCurrent = false;

  public function __construct ($strModelName, $strSQLString = '')
  private function fetch ($strSQLString = '')
  private function populate ($strQuery)
  public function __set ($strName, $strValue)
  public function __call ($strName, $arrArgs)
  public function current ()
  public function key ()
  public function next ()
  public function rewind ()
  public function valid ()
  public function count ()
  public function offsetSet ($mixOffset, $mixValue)
  public function offsetExists ($mixOffset)
  public function offsetUnset ($mixOffset)
  public function offsetGet ($mixOffset)
}//class
