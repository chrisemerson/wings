DirectQuery Class

< Back

The DirectQuery class allows you to connect to the database directly and execute queries on it, bypassing the ORM and Models framework.

Methods & Properties

DirectQuery->__construct () (Constructor)

Opens a new database connection and connects to the database using the information in app.xml.

DirectQuery->query ($strSQL)

Executes a query on the database and returns the results.

DirectQuery->escape_string ($strString)

Escapes a string according to the current database connection.

DirectQuery->affected_rows

Returns the number of affected rows from the last query.

DirectQuery->insert_id

Returns the Autonumber value of the last INSERT operation.

DirectQuery->connect_errno

Ruturns the error number of the connection attempt (if any). If no error occurred, ruturns 0.

DirectQuery->connect_error

Returns the error of the connection attempt (if any).

DirectQuery->errno

Returns the error number of the last query (if any). If no error occurred, returns 0.

DirectQuery->error

Returns the error of the last query (if any).

DirectQuery->close ()

Closes the database connection.
