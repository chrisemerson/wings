BaseController Class

< Back

This class serves as a base for all the controllers of the system, and provides them with their basic functionality. As a result, it is abstract and cannot/should not be instantiated itself. It does not extend any other classes or implement any interfaces.

Basic Structure

abstract class BaseController {
  protected $view = null;

  public final function __construct ()
  public function index ()
  protected function renderView ()
}//class
