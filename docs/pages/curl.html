CURL Class

< Back

This class is a wrapper for the cURL functionality in PHP that provides an object oriented interface to the library. cURL must be installed in order to use this class.

When using this class, the CURLOPT_RETURNTRANSFER option is set to true automatically.

Methods & Properties

CURL->__construct($strURL = '') (Constructor)

Creates a new instance of the CURL class. You can optionally pass in a URL which will set the URL to be requested.

CURL->__destruct() (Destructor)

Closes the curl connection when the object is destroyed using curl_close().

CURL->var

Setting any property on the CURL object sets the option with that name. The full list of options can be found at http://uk.php.net/manual/en/function.curl-setopt.php.

CURL->set($arrOptions)

The set() method allows you to set several options at the same time through an associative array.

CURL->addHeader($strName, $strValue)

This method lets you easily add request headers to the transfer. Later headers of the same name will override earlier ones.

CURL->exec()

Initiates the request using the URL and options provided. It returns the response received.

CURL->errno

After a request, this contains the last error number if any, or 0 if no error occurred.

CURL->error

After a request, this contains the last error (if any).

CURL->var

After a request, getting a property allows you to get information about the last request. The full list of available information can be found at http://uk.php.net/manual/en/function.curl-getinfo.php.
